 js-modernizer
JavaScript Modernization Tool (ES5 → ES6) is a React 18 + Vite app that converts ES5 code to modern ES6 syntax. Features include var → let/const, arrow functions, real-time conversion, syntax highlighting, and a responsive Tailwind CSS interface. Ideal for learning and legacy code modernization.

 JavaScript Modernization Tool (ES5 → ES6)

This React 18 + Vite + Tailwind CSS project automatically converts ES5 JavaScript code to modern ES6 syntax.
Features

- Convert `var` → `let` / `const`
- Function expressions → Arrow functions
- Real-time conversion display
- Syntax highlighting using Prism
- Responsive UI with Tailwind CSS
